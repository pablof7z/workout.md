# Coach screen

A per-exercise side panel (reached by swipe-right, TikTok-style) where you jot how a set felt in plain language and the coach replies terse and dry, then actually mutates the upcoming plan — reducing weight, skipping, deloading, or adding a note. In the mock, the coach is a scripted keyword-driven policy that genuinely edits upcoming steps, as a drop-in point for a real model call later.
