# Block Model for Circuits and Supersets

## Block Model

Circuits and supersets are first-class in the Workout.md UI, represented with a block model containing rounds and an inline movement mini-map (e.g. A1 ▶ A2). <!-- [^42f3c-7f60a] -->

Circuits and supersets are first-class in the Workout.md UI, represented with a block model containing rounds and an inline movement mini-map (e.g. A1 ▶ A2). Rest pages are inserted between rounds but never after the final round, matching the A1, A2, rest, A1, A2, rest, A1, A2 pattern. The mini-map is a horizontal-scrollable row showing the movements in the current block, with the current movement marked with a play.fill SF Symbol. <!-- [^42f3c-5decb] -->
