# Coach Configuration

## Coach as Configurable Policy

The coach in Workout.md is a configurable policy, not a persona. The default coach voice is dry, direct, and sparse. <!-- [^42f3c-54bc5] -->

## Configuration Levels

Coach configuration is adjustable at global, block, methodology, workout, and override levels. <!-- [^42f3c-54bc5] -->

## Configuration as Text

Coach and context settings are written in plain language, editable anytime as configuration-as-text, serving as the knobs behind every AI decision. <!-- [^42f3c-c8101] -->
