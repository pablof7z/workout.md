# Plan Repair

## Core Principle

Plan repair in Workout.md is forward-only: no guilt, no catch-up, and no streaks. When a gap occurs, the plan surface shows a freshly repaired next session instead of red missed-days. The engine is surfaced as a single 'what should I do?' affordance that regenerates the next session from current reality, surfacing the next useful workout. <!-- [^42f3c-7f560] -->
