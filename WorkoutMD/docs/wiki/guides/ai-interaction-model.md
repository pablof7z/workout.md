# AI Interaction Model

## AI Interaction Model

AI is invisible by default and appears on-tap when wanted. There is no chat tab as the main event. Every prescribed number shown in the UI is a ghost target representing an AI decision that looks like a plan, not a bot. When a user misses reps twice or flags pain, a single quiet inline suggestion appears in context as a diff (e.g. "drop set 3 to 115?") with accept, ignore, or override — not a conversation. Text or voice input to the AI returns a proposed change rendered as before/after that the user can accept, not chatter or a wall of encouragement. <!-- [^42f3c-09957] -->
