# Deviation as First-Class Input

## Core Philosophy

Workout.md treats deviations — missed reps, skips, substitutions, pain, fatigue, and missed sessions — as first-class inputs that feed planning. Deviation is the happy path: the plan is disposable and reality is the source of truth. <!-- [^42f3c-e93b5] -->
