# Markdown as Storage Layer

## Markdown Storage and Portability

Markdown is the storage and portability layer for Workout.md, not the UI. Every workout is stored as a clean Markdown file that could be opened in Obsidian, is exportable and syncable, and can be fed to the user's own AI. This provides data ownership and provider flexibility via portable Markdown, with support for hosted or local AI. <!-- [^42f3c-4b622] -->
