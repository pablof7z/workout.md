# Tracking Inputs

## Tracking Inputs

Tracking inputs in Workout.md are structured rich fields, counters, and selectors — never raw Markdown typing. Logging a set takes one or two taps: weight and reps pre-fill from the plan or last session, and the user nudges them if reality differed. A set is logged as 'as planned' with one tap, or 'adjusted' by tapping the number and dialing it. Skipping, substituting, adding a set, and flagging pain or fatigue are all one gesture away, never buried in a menu. <!-- [^42f3c-a68e2] -->
