# Today Screen

## Today Screen

The Today screen is the default surface for 90% of usage. It presents a vertical list of today's exercises, with prescribed targets shown as ghosted placeholders. The distinction between prescribed (ghosted) and actual (solid) set data is always visible on the Today screen. <!-- [^42f3c-c008f] -->
