# Workout Runner Design

## Runner Design

The Workout.md iOS app uses a full-bleed design with no card containers — glass is reserved strictly for floating controls. The workout runner uses a TikTok-style full-screen pager showing one set at a time, advanced via tap or swipe. The background hue shifts per block section (e.g. chest = crimson, superset = purple). <!-- [^42f3c-c2972] -->
