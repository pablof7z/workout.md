# iOS Prototype Overview

## Overview

The Workout.md iOS prototype is a real native SwiftUI app targeting the iOS 26 SDK, not a web emulation. It uses pure SwiftUI with no third-party packages and Swift language mode 5 compatibility. <!-- [^42f3c-53669] -->

## App Flow

The app flow is Today → Runner → Done, with no NavigationStack. <!-- [^42f3c-b46d8] -->

## Accessibility

Accessibility labels and values are set on all interactive controls and the mini-map. <!-- [^42f3c-f39e5] -->
