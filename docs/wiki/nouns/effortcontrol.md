# EffortControl

An expressive, interactive effort input that replaces the old Easy/Moderate/Hard pills. Collapsed it's a compact glass capsule ('Rate effort' or the committed value recolored); tapping morphs it into an interactive glass scale mapping RPE 6–10 with detents and haptics, recording the value into the shared WorkoutSession.
