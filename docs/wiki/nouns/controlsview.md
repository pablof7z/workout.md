# ControlsView

The floating glass control cluster docked to the bottom safe area. Intentionally light after interaction rework: an expressive effort rating, the reps stepper (rep sets only), and a small Skip. No Log & Next (advancing is a swipe down); no Note button. Besides the coach cue pill, the only place in the runner that uses Liquid Glass — reserved for floating controls, never content.
