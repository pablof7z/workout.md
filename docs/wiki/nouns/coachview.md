# CoachView

The scripted mock coach, scoped to the current exercise. Full-bleed dark (same aesthetic as the runner — NOT a sheet, NOT web cards). The user types a plain-language note; a local keyword policy appends a terse, dry coach reply and applies a concrete change to the shared WorkoutSession, shown as a distinct applied-diff line. The coach voice is dry and direct — no pep talk.
