# WorkoutListView

The full workout overview presented as a sheet from the top context strip. Because a sheet is a native surface, a grouped List with rows is the correct idiom here — this is NOT the full-bleed, no-cards runner. Rows are grouped by block, the current step is highlighted, and tapping any row jumps the pager to that step.
