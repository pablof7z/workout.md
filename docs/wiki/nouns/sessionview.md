# SessionView

TikTok-style horizontal pager with two pages: the Coach screen on the LEFT and the runner on the RIGHT, defaulting to the runner. Swiping RIGHT reveals the Coach (scoped to the current exercise); swiping LEFT returns to the runner. Horizontal page swipe and runner's vertical set-paging operate on perpendicular axes so they don't deadlock.
