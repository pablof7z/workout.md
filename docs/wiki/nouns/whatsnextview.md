# WhatsNextView

The 'what should I do next?' surface: a prominent Today entry (outside a running session) that asks the coach to propose or REPAIR the next session from recent history/goals. Forward-only — no guilt, no catch-up, no streaks — so a gap in training is surfaced as 'here's the next useful session,' never as a red missed-days count. Falls back to a deterministic coach-independent repair when no coach provider is reachable.
