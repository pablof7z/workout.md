# TopStripMetrics

Sizing enum for the floating TopContextStrip pill in RunnerView, shared with StepPageView so its topReserve is derived from the pill's actual rendered geometry instead of a guessed constant. Defines topOffset=6, height=34 (caption-weight text + padding), clearance=16, totalReserve=56.
