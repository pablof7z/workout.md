# Data Model and Markdown Storage

## Data Model and Markdown Storage

Workout data is stored as portable Markdown, supporting both hosted and local AI providers. Planned targets and actual logged results are always preserved separately; the AI never edits factual completed logs. <!-- [^42f3c-80a01] -->
