# Coach Configuration and Behavior

## Coach Configuration and Behavior

The coach is configurable policy, not a persona. Its default voice is dry, direct, and sparse. Coach and context settings are configuration-as-text: written in plain language, editable anytime, and serving as the knobs behind every AI decision. Configuration is layered at global, block, methodology, workout, and override levels. When the user talks to the AI—adjusting a plan or asking why—the AI returns a proposed change rendered as a before/after diff the user can accept, not a wall of chatter. <!-- [^42f3c-9c1ba] -->
