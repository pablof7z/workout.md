# Effort Feedback Dial

## Effort Feedback Dial

The user can provide effort feedback easily on the current set. Initially this is a simple Easy/Moderate/Hard selection; later it becomes an RPE 6–10 rating. The effort rating is presented as a single glass 'Rate effort' capsule that morphs into a draggable RPE dial. The dial features a teal-to-amber-to-red gradient track, a live recoloring RPE value, and haptic ticks at each detent (RPE 6–10). Committing the set is done via an amber Set commit button. The dial's drag gesture is a .highPriorityGesture so it wins over the page swipe. <!-- [^42f3c-089d3] -->

## Reps Stepper

The reps number and +/− stepper buttons are large and bold. The reps value is displayed at 34pt bold rounded, and the stepper buttons are 52pt circular glass. <!-- [^42f3c-2b502] -->
