# Agent Workflow and Roles

## Agent Workflow and Roles

Coding is done by Sonnet agents, device testing by Haiku agents, and product-readiness is gated by an Opus agent judge. The epic is worked iteratively: each capability lands on its own branch as a PR, is tested, then merged to master. <!-- [^42f3c-252b9] -->
