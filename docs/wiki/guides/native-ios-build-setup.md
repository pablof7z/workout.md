# Native iOS Build Setup

## Project Foundation

The prototype is built as a real native iOS SwiftUI app targeting the iOS 26 SDK with real Liquid Glass APIs, not a web emulation. The project repo is github.com:pablof7z/workout.md on the master branch, with gh authed as pablof7z. <!-- [^42f3c-a00f3] -->

## XcodeGen Project Configuration

The app's project uses XcodeGen with a project.yml, bundle id com.workoutmd.prototype, portrait orientation, iOS 26 target, and no signing for simulator builds. <!-- [^42f3c-149d2] -->

## Deployment

The app deploys to the user's physical iPhone when done, ensuring a polished and sleek finished product. <!-- [^42f3c-a73c0] -->
